========================================
API ENDPOINTS (MOBILE APP USES THESE)
========================================

Authentication:
  POST /api/login          - Get JWT token
  POST /api/logout         - Logout
  GET  /api/me             - Get user profile

Games:
  GET  /api/games          - List all games
  GET  /api/category       - Game categories
  GET  /api/jackpots       - Jackpot info

User Data:
  POST /api/me/balance     - Get current balance
  GET  /api/me/refund      - Refund history

Transactions:
  GET  /api/credits        - Transaction history

========================================
WEBSOCKET SERVERS (REAL-TIME GAME DATA)
========================================

Slots:   wss://cashout.realconnect.online:22150/slots
Arcade:  wss://cashout.realconnect.online:22180/arcade
Server:  wss://cashout.realconnect.online:22190/server

These connect to database directly for real-time game updates.

========================================
KEY DATABASE TABLES
========================================

w_users          - User accounts, balances, authentication
w_games          - Game catalog (slots, arcade, fish games)
w_game_log       - Game play history and results
w_credits        - User transaction history
w_sessions       - Active user sessions
w_api_keys       - API keys for mobile app access
w_game_bank      - Slot game jackpots
w_fish_bank      - Arcade game jackpots

========================================
DATA FLOW EXAMPLE
========================================

1. User Login:
   Mobile App → POST /api/login
              → Laravel queries w_users table
              → Returns JWT token + user data
              → Mobile app stores token

2. Get Balance:
   Mobile App → GET /api/me/balance (with JWT token)
              → Laravel queries w_users.balance
              → Returns current balance

3. Play Game:
   Mobile App → WebSocket connect
              → WebSocket server queries database
              → Game updates w_game_log, w_users.balance
              → Real-time updates sent to mobile app
